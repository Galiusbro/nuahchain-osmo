## ⚠️ Ограничения API

### Лимиты запросов
- **Бесплатный план**: 8 запросов в минуту
- **Превышение лимита**: Ошибка 429 "You have run out of API credits"

### Рекомендации по использованию
1. **Добавляйте задержки** между запросами (минимум 10 секунд)
2. **Приоритизируйте данные** - получайте только самые важные символы
3. **Рассмотрите платный план** для массового получения данных

## 🛠️ Созданные инструменты

### 1. `check_price.sh` - Проверка одного символа
```bash
./check_price.sh EUR/USD
./check_price.sh BTC/USD
./check_price.sh AAPL
./check_price.sh XAU/USD
```

### 2. `get_market_data_optimized.sh` - Массовое получение данных
- Получает основные валютные пары, золото, акции и криптовалюты
- Включает задержки для соблюдения лимитов API
- Сохраняет результаты в JSON файл

### 3. `get_market_data.sh` - Полное получение данных
- Получает максимальное количество символов
- Без задержек (может превысить лимиты)

## 🔧 Технические детали

### API Endpoint
```
https://api.twelvedata.com/time_series
```

### Параметры запроса
- `symbol`: Символ для получения данных
- `interval`: Интервал (1day, 1hour, 1min, etc.)
- `apikey`: API ключ
- `outputsize`: Количество записей (1 для последней цены)

### Формат ответа
```json
{
  "meta": {
    "symbol": "EUR/USD",
    "interval": "1day",
    "currency_base": "Euro",
    "currency_quote": "US Dollar",
    "type": "Physical Currency"
  },
  "values": [
    {
      "datetime": "2025-10-21",
      "open": "1.1641",
      "high": "1.16444",
      "low": "1.16403",
      "close": "1.16424"
    }
  ],
  "status": "ok"
}
```
